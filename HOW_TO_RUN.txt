# How to run the Menopa-AI Web App

This application consists of a Python Flask backend and a React/Vite frontend. You will need to run both concurrently in separate terminal windows.

## 1. Running the Backend

Open a new terminal window and run the following commands:

1. Navigate to the backend directory:
   cd backend

2. Activate the virtual environment:
   .\venv\Scripts\activate
   (Note: If you are using a different environment or venv313, activate that instead like .\venv313\Scripts\activate)

3. Install the dependencies (if you haven't already):
   pip install -r requirements.txt

4. Start the backend Flask server:
   python app.py

The backend server should now be running, typically on http://127.0.0.1:5000.

## 2. Running the Frontend

Open a second, separate terminal window and run the following commands:

1. Navigate to the frontend directory:
   cd frontend

2. Install the necessary Node dependencies:
   npm install

3. Start the development server:
   npm run dev

The frontend will start up and provide you with a local URL (e.g. http://localhost:5173/) to view the web app in your browser.
